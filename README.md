# A2A Multi-Agent Protocol Project

A project demonstrating agent-to-agent (A2A) communication where multiple agents can talk to each other via an execution server.

## Features
- Multiple agents (e.g., Ambani, Modi)
- Async agent executor
- Tool-based availability lookup

## Setup
1. Create virtual env
2. Install requirements
3. Add .env with API keys
4. Run the agent executor

## Run
Example:
PYTHONPATH="$(pwd)" python -m ambani_agent.agent_executor
